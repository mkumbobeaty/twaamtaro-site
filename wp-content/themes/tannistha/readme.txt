== Description ==

Tannistha is a clean, blog WordPress theme. Tannistha features:

* There's a primary color option which can be set from the Customizer.
* There's a secondary color option which can be set from the Customizer.
* Three blog feed views (Excerpt Only, Small Image on Left Excerpt on Right, Large Image and Full Content)
* Multiple options for banners
* Ability to turn sidebar on or off on pages and posts
* Front page banner
* Single blog page banner
* Note: The site tagline appears on the home/front page only on top of the header image/banner

For a complete list of features and detailed documentation, please go to https://www.dualcube.com/tannistha/
For support: admin@dualcube.com
